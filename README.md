# Loja de Carros Java

Sistema web desenvolvido em Java para simular uma loja de carros utilizando estrutura de dados do tipo PILHA (LIFO) para controle do histórico de páginas acessadas.

## Funcionalidades

- Listagem de veículos
- Visualização de detalhes dos carros
- Histórico de páginas visitadas
- Simulação do botão voltar
- Controle de navegação utilizando pilha

## Estruturas de Dados Utilizadas

- Deque
- ArrayDeque
- Pilha (LIFO)

## Tecnologias

- Java
- HttpServer
- HTML
- CSS

## Como executar

Execute a classe:

LojaCarrosWeb.java

Depois acesse no navegador:

http://localhost:9091

## Objetivo do desafio

Aplicar o conceito de PILHA (LIFO — Last In, First Out) em um sistema web Java simulando o histórico de navegação de páginas.
